# Introduction to Deep Learning for Images Classification & Objects Detection

## Training Objectives

Unlike AI systems such as chatbots, which rely on complex algorithms and remote infrastructure, this training offers a simple and progressive approach to Deep Learning applied to computer vision, focusing in particular on image classification and object detection in images.

Participants will learn, step by step, to:

- understand how a neural network works,
- build a neural network model dedicated to images classification,
- train this model using a database of labeled images,
- evaluate its performance and improve it.

In the second part, the training demonstrates how to go further without starting from scratch, by reusing (transfer learning) already trained complex models to learn how to locate and identify specific objects in images.

The exercises are performed in Python, using the TensorFlow and Keras modules, within a dedicated Python Virtual Environment. Using participants’ personal computers makes learning concrete and reproducible.

---

# Targeted Skills

Upon completion of this training, participants will be able to:

- Manage a Python Virtual Environment dedicated to a Python project using the `uv` manager.
- Program a neural network (dense, then convolutional) for images classification using the Python modules TensorFlow and Keras.
- Prepare a dataset within a supervised learning framework.
- Configure a training and validation pipeline.
- Analyze model results.
- Use transfer learning to train pre-trained networks to detect custom objects in images.

---
